# CHANGAYEES UX STRATEGY

Version: 1.0

Status: Foundation Document

Owner: Product & UX Team

---

# 1. UX VISION

Changayees is not a traditional wholesale website.

Changayees is a procurement platform that enables institutional buyers to:

* Discover products
* Submit requirements
* Request quotations
* Communicate through WhatsApp
* Track production progress
* Receive automated updates

The experience should feel:

* Professional
* Trustworthy
* Fast
* Modern
* Mobile Native
